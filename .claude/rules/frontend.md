# Frontend rules
- Serious intelligence-product look: cream background, charcoal text, dark teal accent, thin borders, information-dense. No purple gradients, glassmorphism, chat bubbles, fake metrics or decorative charts.
- Target cards answer: why this account / why now / signal strength / who owns it / what proves it / what is uncertain. Strongest evidence first; caveats never hidden.
- Escape all dynamic HTML (server/render.ts esc()); external links rel="noopener noreferrer nofollow".
